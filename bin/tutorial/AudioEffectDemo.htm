Audio Effect Demo

Developed by Dr. Andrew Kwok-Fai LUI

Learning Objective: Describe the effect of audio processing operations on audio signals

First load an audio file using the Audio File Selector. Press Play and Stop button to control the playing of hte audio.

There are many Audio Processing operations available in the Audio Effect Control panel. Press any button labeled with an Audio Processing operation to turn it on/off. Play the audio file to listen to the effect of the audio operation.
